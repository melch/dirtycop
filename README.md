# Dirty::Cop

## Credit
A modification of skanev's gist, original found here: https://gist.github.com/skanev/9d4bec97d5a6825eaaf6



## Installation

```ruby
gem 'dirty-cop'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dirty-cop

## Usage

```shell
$ dirty
```
